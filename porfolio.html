Sobre Mi

Desde chico, siempre quise ser programador porque me encanta la tecnología. Aunque soy full stack, me siento más cómodo y disfruto más trabajando en el back-end.

Me motivan los desafíos y aprender nuevas formas de resolver cada problema que se presenta en los proyectos. Soy metódico, presto mucha atención a los detalles y prospero en entornos dinámicos y bajo presión, manteniendo un excelente trabajo en equipo.

Como persona, me considero motivado y dedicado a lo que hago. Siempre busco explorar nuevas tecnologías para seguir mejorando y mantenerme actualizado en este campo que tanto me apasiona.

Tecnologias

TypeScript

Un lenguaje de programación fuertemente tipado que se construye sobre JavaScript.

JavaScript

Un lenguaje de programación dinámico utilizado tanto en desarrollo frontend como backend, esencial para crear interactividad en la web.

CSS

Un lenguaje de estilo en cascada utilizado para describir la presentación de un documento escrito en HTML o XML, proporcionando diseño visual a las páginas web.

PostgreSQL

Un sistema de gestión de bases de datos relacional y de código abierto, conocido por su estabilidad, extensibilidad y soporte para estándares SQL avanzados.

HTML

El estándar de marcado utilizado para estructurar el contenido en la web, creando la base para la construcción de páginas y aplicaciones web.

Node.js

Un entorno de ejecución para JavaScript basado en el motor V8 de Chrome, utilizado principalmente para construir aplicaciones backend escalables.

NestJs

Un framework progresivo para Node.js que ayuda a crear aplicaciones del lado del servidor eficientes y escalables con TypeScript.

Cloudinary

Una plataforma de gestión de imágenes y videos en la nube que ofrece soluciones para almacenamiento, manipulación y entrega optimizada de medios.

Proyectos

ComiCraft

ComiCraft es una APP WEB donde los usuarios pueden registrarse para leer sus cómics favoritos dentro de la app adquiriendo una suscripción mensual o anual. También ofrece la posibilidad de subir sus propios cómics para que los lea la comunidad.

Pedia

Pedia es una aplicación web para gestionar turnos con un podólogo. Permite a los usuarios registrarse, agendar y cancelar citas de manera sencilla. Además, ofrece información sobre la podología y cómo reconocer problemas comunes en los pies, asegurando una atención especializada y cómoda a través de una interfaz intuitiva.

M4Proyecto

M4Proyecto es un backend de e-commerce diseñado para vender productos de Apple. Ofrece funcionalidades esenciales como registro de usuarios, creación de órdenes de compra, gestión de productos e integración de imágenes con Cloudinary. Este proyecto fue desarrollado como parte de un módulo de un bootcamp.
